# JS-O.mok  금수 규칙

## 1. 개요

사용자 설정을 통해 흑과 백 각각에 대해 **금수(Forbidden Moves)** 규칙을 독립적으로 적용할 수 있는 유연한 시스템을 갖추고 있습니다.

## 2. 주요 금수 규칙 🚫

### 3-3

두 개 이상의 **열린 삼(Open Three)**이 동시에 만들어지는 지점에 착수할 수 없습니다.

* **연속 삼**: `_XXX_`와 같이 세 개의 돌이 나란히 놓인 형태.
* **뛰어 삼 (Jumped Three)**: `_X_XX_` 또는 `_XX_X_`와 같이 한 칸의 빈 공간을 포함하여 삼을 형성하는 정밀 패턴을 포함합니다.

### 4-4

두 개 이상의 **사(Four)**가 동시에 만들어지는 지점에 착수할 수 없습니다.

* 상대방이 막지 않으면 다음 수에 바로 승리(오목)가 되는 패턴이 두 방향 이상 형성되는 경우입니다.

### 육목

같은 색상의 돌이 **6개 이상** 연속으로 놓이는 것을 금지합니다.

* 금수 옵션 활성화 시: 육목을 만드는 위치에 착수할 수 없습니다.
* 금수 옵션 비활성화 시: 6개 이상의 돌이 놓여도 승리로 인정됩니다.

---

## 3. 사용자 제어 기능

컨트롤 패널을 통해 게임의 난이도와 규칙을 실시간으로 조정할 수 있습니다.

* **흑/백 금수 개별 설정**: 흑과 백 각각의 금수 적용 여부를 체크박스로 선택 가능합니다.
* **디버그 가이드라인**: 금수 지점 착수 시도 시 붉은색 선으로 판정 근거를 시각화합니다.
* **AI 설정**: 봇의 수비 비중과 탐색 깊이를 조절하여 대국 난이도를 최적화합니다.
